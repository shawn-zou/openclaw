## 💡 使用技巧

### 1. 善用搜索
在文档中使用 `Ctrl+F` (Windows) 或 `Cmd+F` (Mac) 快速查找关键词。

### 2. 做笔记
准备一个笔记本，记录重点和疑问。

### 3. 多实践
不要只看文档，要动手写代码。

### 4. 提问题
遇到问题先思考，再查文档，最后问社区。

### 5. 常复习
定期回顾已学内容，加深理解。
